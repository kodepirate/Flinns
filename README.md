## 📖 About The Project

**Flinns NFT** is not just a landing page; it's a visceral, scroll-driven journey to the depths of the ocean. Built for the modern web, this project aggressively optimizes performance while delivering a stunning, physics-based 3D interactive experience. Whether you're showcasing an NFT collection to prospective buyers or simply looking to blow minds, Flinns delivers.

## ✨ Key Features

- 🤿 **Seamless Deep-Sea Descent:** Perfectly blended vertical gradients across all sections, creating an uninterrupted visual journey from the ocean surface to the trench floor.
- 🧊 **3D Interactive Hover Physics:** Custom glassmorphic cards and elements that track mouse movement in 3D space, tilting and glowing dynamically.
- ⚡ **Blazing Fast Performance:** Aggressively optimized for speed. Utilizes React `Suspense` and `lazy` loading for code-splitting, offloading secondary visual components to reduce initial main-thread blocking. Layout recalculation loops are eliminated through bounding-box caching.
- 📜 **Scroll-Driven Animations:** A central roadmap timeline that organically "fills" and animates as the user scrolls down the page.
- 📱 **Fully Responsive:** Thoughtfully adapted and tested for pixel-perfect display on mobile, tablet, and desktop devices.

## 🚀 Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

Ensure you have the latest version of Node.js and npm installed.

```bash
npm install npm@latest -g
```

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/kodepirate/Flinns.git
   ```
2. **Navigate into the project directory:**
   ```bash
   cd Flinns
   ```
3. **Install the dependencies:**
   ```bash
   npm install
   ```
4. **Start the development server:**
   ```bash
   npm run dev
   ```
   > 💡 Open your browser and navigate to `http://localhost:5173` to see the site in action!

## 🏗️ Production Build

To test the highly-optimized production build and measure true Lighthouse performance scores:

1. **Build the production bundle:**
   ```bash
   npm run build
   ```
2. **Preview the production build locally:**
   ```bash
   npm run preview
   ```
